# 밴드 성적 자동 채점기 — 사용 안내

이 문서는 조교가 교수님께 전달하는 설치·실행 안내입니다. 컴퓨터 전문 지식이 없어도
따라 하실 수 있도록 순서대로 정리했습니다.

## 1. 설치

1. 조교가 알려준 다운로드 링크(GitHub Release)에서 설치 파일(`.exe`)을 받습니다.
2. 다운로드한 파일을 더블클릭합니다.
3. 이때 **"Windows에서 PC를 보호했습니다"** 라는 파란색 경고 창이 뜰 수 있습니다. 이건
   바이러스가 아니라, 아직 많은 사람이 써보지 않은 프로그램이라 Windows가 일단 조심하라고
   띄우는 안내입니다. 아래처럼 하면 됩니다:
   - 창 안의 **"추가 정보"** 를 클릭
   - 아래에 나타나는 **"실행"** 버튼 클릭
4. 설치가 자동으로 진행되고, 끝나면 바탕화면과 시작 메뉴에 **"밴드 성적 자동 채점기"**
   아이콘이 생깁니다. 별도로 뭘 누를 필요 없이 여기서 끝입니다.

## 2. 첫 실행 — 설정 입력

1. 바탕화면의 "밴드 성적 자동 채점기" 아이콘을 더블클릭합니다.
2. 처음 실행하면 설정 파일이 아직 없다는 안내 창이 뜨고 프로그램이 종료됩니다. 이때
   `%APPDATA%\밴드 성적 자동 채점기\input\`(보통 `C:\Users\사용자명\AppData\Roaming\밴드 성적 자동 채점기\input\`) 폴더가 자동으로 만들어지고, 그 안에
   `1_설정.xlsx` 파일이 함께 생성됩니다.
3. 그 엑셀 파일을 열어 노란 칸(측정 시작일·종료일·총점 상한·대상 밴드 등)을 채우고
   저장합니다. 궁금한 항목은 엑셀 안의 "설명" 칸을 참고하시거나 조교에게 문의하세요.
4. 다시 아이콘을 더블클릭해 프로그램을 실행합니다.

## 3. 실행 — 무엇을 하시겠습니까?

설정이 정상이면 아래 3가지 중 하나를 고르는 창이 뜹니다.

- **데이터 수집 후 채점**: 처음 실행하거나, 밴드에 새 게시글/댓글이 올라와서 다시
  전체를 가져와야 할 때 선택하세요. 밴드 로그인 창이 뜨면 **직접** 로그인(2단계 인증
  포함)해야 합니다 — 자동 로그인은 지원하지 않습니다. 로그인 후에는 자동으로 진행됩니다.
  밴드 내용이 많으면 시간이 꽤 걸릴 수 있습니다.
- **채점만 다시 실행**: 데이터는 이미 받아놓은 상태에서, 로스터 학번 입력이나 예외
  처리 엑셀만 고쳐서 점수를 다시 계산하고 싶을 때 선택하세요. 밴드 로그인이 필요 없고
  훨씬 빠릅니다.
- **취소**: 아무 것도 하지 않고 종료합니다.

## 4. 결과 확인

채점이 끝나면 완료 안내 창이 뜨고(파일 경로도 그대로 보여줍니다), 결과 파일은
`%APPDATA%\밴드 성적 자동 채점기\out\`(보통 `C:\Users\사용자명\AppData\Roaming\밴드 성적 자동 채점기\out\`)
폴더에 생깁니다. 탐색기 주소창에 `%APPDATA%`를 입력하면 바로 이동할 수 있습니다:

- `결과_및_감사_....xlsx` — 학생별 점수와, 왜 그 점수가 나왔는지 확인할 수 있는 감사
  자료가 함께 있는 파일입니다. 성적 이의제기가 들어오면 이 파일로 근거를 확인하시면
  됩니다. "점수"(활동 참여 점수)와 "과제점수"(과제글에 댓글로 제출한 개수 기준)는
  서로 더해지지 않는 별도 컬럼입니다.
- `scores_....csv` — 점수만 담은 파일입니다.

## 5. 문제가 생겼을 때

프로그램이 뭔가 잘못됐다고 판단하면 화면에 한국어로 원인과 해야 할 일을 안내하는 창을
띄웁니다(예: "설정 확인 필요 — 밴드 URL을 입력해주세요" 등). 안내대로 하시면 대부분
해결됩니다. 안내 문구로 해결이 안 되거나, "예상치 못한 오류" 창이 뜨면 그 창의 내용을
캡처해서 조교에게 전달해 주세요.
