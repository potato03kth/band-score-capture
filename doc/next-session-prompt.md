## 이번 세션 할 일 — Phase 4: 결과 시트 0값 하이라이트

1. 대상: `score/csv.js`의 `writeAuditWorkbook` 함수 안, `resultSheet`를 렌더링하는 부분.
2. 결과 시트에서 다음 4개 컬럼 값이 0인 셀에 배경 스타일(노란색 등, 기존 `lib/xlsx.js`의
   `applyYellow` 헬퍼 스타일과 일관되게)을 적용한다: **활동일수, 게시글수, 댓글수, 점수**
   (Phase 3에서 COLUMN_DESCRIPTIONS에 이미 추가된 컬럼들 — `score/csv.js` 현재 헤더 순서:
   학번, 실명, user_no, band_id, band_name, 활동일수, 게시글수, 댓글수, 점수, 매핑상태,
   수집완료여부, 감사근거, 산출시각).
3. ExcelJS로 셀에 배경을 줄 때는 `cell.fill = { type: 'pattern', pattern: 'solid',
   fgColor: { argb: 'FF...' } }` 형태를 쓴다. `lib/xlsx.js`에 이미 쓰는 노란색 ARGB 값이 있는지
   먼저 확인하고 있으면 재사용해라(색상 일관성).
4. **주의: "0이 곧 문제"가 아니다.** 활동이 아예 없는 학생(정상적으로 0점)과, 데이터 수집이
   덜 됐거나 매핑이 안 돼서 0으로 보이는 경우를 구분하려는 목적이 아니라 — "교수가 결과를 훑어볼
   때 0을 놓치지 않도록" 시각적으로 두드러지게 하는 것이 목적이다. 로직을 복잡하게 만들지 말고
   단순히 "해당 셀 값이 0이면 배경 강조"로 구현해라.

## 검증

  Phase 4 검증 기준과 동일 — "워크북을 다시 읽어 해당 셀 `fill` 세팅 확인").
